⏳ O QUE VOCÊ PRECISA FAZER (MANUAL):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

FASE 1 — RENDER (10 min)
  1. Acesse: https://dashboard.render.com
  2. Clique "New +"
  3. Selecione "Web Service"
  4. Conecte repositório GitHub: crediclass-dashboard-grupos
  5. Configure:
     • Name: crediclass-dashboard
     • Runtime: Python 3
     • Build Command: pip install -r backend/requirements.txt
     • Start Command: uvicorn backend.main:app --host 0.0.0.0 --port $PORT
     • Plan: Standard ($12/mês)
  6. Clique "Create Web Service"
  7. Aguarde build (2-3 minutos)
  8. Vá para: Environment
  9. Adicione variáveis (veja backend/.env.example):
     • GOOGLE_SHEETS_ID
     • GOOGLE_API_KEY
     • PIPERUN_API_KEY
     • ENVIRONMENT=production
     • DEBUG=false
  10. Render faz redeploy automático

FASE 2 — CLOUDFLARE (5 min)
  1. Acesse: https://dash.cloudflare.com
  2. Selecione domínio: csrtecnologia.com.br
  3. Vá para: DNS → Records
  4. Crie CNAME:
     • Name: crediclass
     • Type: CNAME
     • Content: onrender.com
     • Proxy: Proxied (laranja)
     • TTL: Auto
  5. Save
